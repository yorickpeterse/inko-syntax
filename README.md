# inko-syntax

A simple syntax highlighting library for [Inko](https://inko-lang.org), with
[Pygments](https://pygments.org/) compatible HTML output.

# Requirements

- Inko `main` (for now)

# Installation

```bash
inko pkg add github.com/yorickpeterse/inko-syntax 0.1.0
inko pkg sync
```
